General Controls:
View controls:
Numpad 8 2 4 6 - Move view.
Numpad 1 + 8 2 4 6 - Move view faster.
Numpad 3 + 8 2 4 6 - Move view even faster.
Numpad 9 - Allow to move view outside the level boundaries.

Switch menu:
0 - Save Project
1 - Main menu
2 - Geometry editor
3 - Tile editor
4 - Camera editor
5 - Light editor
6 - Room size
7 - Effect editor
8 - Prop editor
9 - Environment editor

Other:
Tab + Shift - Minimize the editor
Escape (VANILLA) / Shift + Escape (DROUGHT) / Tab + X + Enter (DRY) - Close the editor
Tab (VANILLA)  / Tab + Z + R (DROUGHT) / Tab + X + C (DRY) - Abort rendering
